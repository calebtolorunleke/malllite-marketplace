🛍️ MallLite – Modern E-Commerce Web Application

A fully responsive, modern e-commerce frontend built with React and Tailwind CSS, designed to simulate a real-world online shopping experience with routing, cart management, product filtering, and checkout flow.

🚀 Live Demo
🌐 https://malllite-marketplace.vercel.app/

✨ Features
🛒 Shopping Experience
Dynamic product listing from API
Add to cart with quantity management
Remove items from cart
Real-time cart total calculation
Persistent cart UI (drawer system)

🔍 Product Discovery
Search functionality
Category filtering
Pagination system (8 products per page)
Loading skeleton states

📄 Routing & Pages
Home page (Hero + preview products)
Products page (full catalog)
Product detail page (/product/:id)
Checkout page with order summary

🎯 UI/UX
Fully responsive design (mobile-first)
Smooth transitions and hover effects
Modern ecommerce layout
Clean Tailwind-based styling system

🧠 Tech Stack
React.js
React Router DOM
Tailwind CSS
JavaScript (ES6+)
Fake Store / REST API (or custom API)

⚙️ Installation & Setup

Clone the repository:

git clone https://github.com/calebtolorunleke/malllite.git

Install dependencies:

npm install

Run development server:

npm run dev
